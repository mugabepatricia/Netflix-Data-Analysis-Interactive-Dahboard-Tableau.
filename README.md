# Netflix Data Analysis & Interactive Dashboard (Tableau)
## Project Overview
This project explores a Netflix dataset to uncover content distribution patterns, trends, and key insights using Tableau.
The goal was to transform raw data into an interactive and visually compelling dashboard that allows users to explore Netflix’s catalog across different dimensions such as country, genre, rating, and time.
## Dataset Description
The dataset contains information on Netflix Movies and TV Shows, including:
- Title
- Type (Movie or TV Show)
- Country
- Genre
- Rating
- Release Year
- Date Added
- Duration
## Project Objectives
	•	Analyze the distribution of Netflix content globally
	•	Identify trends in content production over time
	•	Explore genre popularity and ratings distribution
	•	Build an interactive dashboard for dynamic exploration
## Data Visualization & Analysis
The following visualizations were created in Tableau:
- Total Movies & TV Shows by Country (Map)
	•	Displays the geographic distribution of Netflix content
	•	Highlights countries with the highest content production
- Ratings Distribution (Bar Chart)
	•	Shows the frequency of content ratings
	•	Helps identify the most common audience categories
- Movies vs TV Shows Distribution (Bubble Chart)
	•	Visual comparison of content types
	•	Provides a quick overview of catalog composition
- Top 10 Genres (Column Chart)
	•	Identifies the most popular genres on Netflix
	•	Helps understand audience preferences
- Total Content by Year
	•	Tracks content release trends over time
	•	Highlights growth patterns in Netflix’s catalog
## Interactive Dashboard Features
The Tableau dashboard includes dynamic filters and KPIs for deeper exploration:
- Filters:
	•	Title
	•	Type (Movie / TV Show)
- KPIs:
	•	Date Added
	•	Release Year
	•	Genre
	•	Duration
	•	Rating
These features allow users to interact with the data and drill down into specific insights.
